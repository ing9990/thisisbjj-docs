# 출석 통계

출석 데이터를 다양한 관점에서 분석합니다.

---

{% hint style="warning" %}
출석 통계는 [PARTNER 구독](../subscription.md)이 필요합니다.
{% endhint %}

## 통계 항목

| 항목 | 설명 |
|---|---|
| **월별 출석 추이** | 6개월간 월별 총 출석 횟수 변화 |
| **요일별 분포** | 어떤 요일에 출석이 많은지 |
| **시간대별 분포** | 어떤 시간에 출석이 집중되는지 |
| **벨트별 출석률** | 벨트 색상별 평균 출석률 |

## 회원별 월간 출석

특정 회원의 월간 출석 현황을 달력 형태로 확인합니다.

- [회원 상세](../members/detail.md) 페이지에서 개별 회원의 월간 출석을 조회할 수 있습니다.
- 연속 출석(스트릭), 월간 출석률 등이 표시됩니다.

## 활용 예시

| 상황 | 활용 방법 |
|---|---|
| 수업 시간표 조정 | 시간대별 분포를 확인하여 인기 시간대에 수업 추가 |
| 회원 리텐션 개선 | 벨트별 출석률로 이탈 위험 벨트 그룹 파악 |
| 시설 운영 효율화 | 요일별 분포로 피크 요일 파악, 청소/정비 일정 조정 |
