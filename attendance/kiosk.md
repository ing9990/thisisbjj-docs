# 키오스크 설정

도장 입구에 태블릿이나 PC를 설치하여 **회원 셀프 체크인** 시스템으로 활용합니다.

---

{% hint style="warning" %}
키오스크를 사용하려면 [지도자 인증](../getting-started/leader-verification.md)이 완료되어야 합니다.
{% endhint %}

## 키오스크 세팅

1. 태블릿 또는 PC에서 [주짓주짓 매니저](https://manager.thisisbjj.com)에 로그인합니다.
2. **출석** → **키오스크** 메뉴로 이동합니다.
3. 브라우저를 **전체 화면(F11)** 으로 설정하면 깔끔한 키오스크 화면이 됩니다.

{% hint style="info" %}
별도 전용 기기가 필요하지 않습니다. 기존에 보유한 태블릿이나 PC를 활용하세요.
{% endhint %}

## 체크인 과정

```
① 휴대폰 뒷자리 4자리 입력 → ② 회원 확인 → ③ 체크인 완료
```

### ① 전화번호 입력

회원이 **휴대폰 뒷자리 4자리**를 입력합니다.

### ② 회원 확인

- 해당 번호의 회원이 1명이면 바로 확인 화면이 나타납니다.
- 같은 뒷자리를 가진 회원이 여러 명이면 **선택 화면**이 나타납니다.

### ③ 체크인 완료

체크인 확인 화면이 나타나고 출석이 기록됩니다.

- **횟수제 회원권**은 체크인 시 **자동으로 1회 차감**됩니다.
- **기간제 회원권**은 차감 없이 출석만 기록됩니다.

## 벨트 색상 설정 (선택)

[대시보드](../dashboard.md)에서 **"출석 시 벨트 설정"** 옵션을 켜면, 벨트가 미설정된 회원이 체크인할 때 **벨트 색상 선택 화면**이 나타납니다.

- 원형 컬러 피커로 직관적으로 벨트를 선택합니다.
- 이미 벨트가 설정된 회원은 이 화면이 나타나지 않습니다.
- 이 기능은 [벨트 관리](../members/belt-management.md)의 "초기 벨트 설정"과 같은 효과입니다.

## 키오스크 운영 팁

| 팁 | 설명 |
|---|---|
| 전체 화면 | 브라우저 F11로 전체 화면 설정 |
| 자동 잠금 해제 | 태블릿의 자동 잠금(화면 꺼짐)을 비활성화하세요 |
| 충전 | 태블릿을 항상 충전 상태로 유지하세요 |
| 위치 | 회원이 입장할 때 자연스럽게 접근할 수 있는 위치에 설치하세요 |
