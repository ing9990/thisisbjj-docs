# 출석 & 키오스크

태블릿이나 PC를 활용하여 회원 셀프 체크인 시스템을 운영합니다.

---

{% hint style="warning" %}
출석 기능은 [지도자 인증](../getting-started/leader-verification.md)이 완료되어야 사용할 수 있습니다.
{% endhint %}

## 기능 구성

| 기능 | 설명 | 요금제 |
|---|---|---|
| [키오스크 설정](kiosk.md) | 셀프 체크인 시스템 | FREE |
| 오늘의 출석 | 오늘 체크인한 회원 실시간 확인 | PARTNER |
| [출석 통계](stats.md) | 월별/요일별/시간대별 분석 | PARTNER |

## 오늘의 출석

[주짓주짓 매니저](https://manager.thisisbjj.com) → **출석** 메뉴의 첫 화면입니다.

- 오늘 체크인한 회원 목록과 총 출석 수를 실시간으로 확인합니다.
- 키오스크 체크인 / 앱 체크인 구분이 표시됩니다.

{% hint style="info" %}
오늘의 출석 상세 조회는 [PARTNER 구독](../subscription.md)이 필요합니다.\
FREE 요금제에서는 [키오스크](kiosk.md) 체크인만 사용할 수 있습니다.
{% endhint %}

## 상세 기능

- [키오스크 설정](kiosk.md) — 셀프 체크인 시스템 세팅 방법
- [출석 통계](stats.md) — 출석 데이터 분석
