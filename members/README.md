# 회원 관리

도장의 모든 회원을 조회하고 관리합니다.

---

{% hint style="warning" %}
회원 관리 기능은 [지도자 인증](../getting-started/leader-verification.md)이 완료되어야 사용할 수 있습니다.
{% endhint %}

## 회원 목록

[주짓주짓 매니저](https://manager.thisisbjj.com) → **회원 관리** 메뉴에서 전체 회원 목록을 확인합니다.

### 검색 & 필터

| 필터 | 설명 |
|---|---|
| 이름/닉네임 검색 | 키워드로 회원 검색 |
| 벨트 색상 | 특정 벨트 회원만 조회 |
| 연령대 | 10대, 20대, 30대 등 |
| 만료 임박 | N일 이내 만료 예정 회원 |
| 회원권 유형 | 기간제 / 횟수제 |

### 정렬

이름순, 잔여일순, 가입일순 등으로 정렬할 수 있습니다.

### 회원 등록

회원이 주짓주짓 앱/웹에서 도장을 검색하여 가입하면 **자동으로 회원 목록에 표시**됩니다.\
또는 [계약서 발송](../contracts/sending.md)을 통해 신규 회원을 등록할 수 있습니다.

{% hint style="info" %}
기존 시스템의 회원 데이터를 한꺼번에 가져오려면 [데이터 이관](../data-migration.md)을 이용하세요.
{% endhint %}
