# 벨트 관리

회원의 벨트를 관리합니다. 개별 승급과 이력 조회는 여기서, 일괄 승급은 [승급식](../ceremonies.md)에서 진행합니다.

---

## 원클릭 승급

[회원 상세](detail.md) 페이지에서 **"승급"** 버튼을 클릭하면 현재 벨트에서 다음 벨트로 즉시 승급합니다.

- 승급 날짜를 지정할 수 있습니다.
- 메모를 남길 수 있습니다 (예: "2024 승급식", "출석 우수").

## 초기 벨트 설정

타 도장에서 이적한 회원 등 벨트가 미설정된 경우, **"초기 벨트 설정"** 으로 현재 벨트를 지정합니다.

{% hint style="info" %}
[데이터 이관](../data-migration.md) 후 또는 신규 관원 등록 후, 벨트를 일일이 등록하기 번거롭다면 [키오스크](../attendance/kiosk.md) 체크인 시 자동으로 벨트를 설정할 수 있습니다. [대시보드](../dashboard.md)의 **"출석체크로 벨트설정하기"** 토글을 켜세요.

<img src="../.gitbook/assets/대시보드 토글.png" alt="대시보드 벨트 설정 토글" width="480">

벨트가 미설정된 회원이 체크인하면 **1회만** 벨트 선택 화면이 나타납니다.

<img src="../.gitbook/assets/출석체크 벨트설정 기능.gif" alt="출석 시 벨트 설정" width="480">
{% endhint %}

## 승급 이력

해당 회원의 전체 벨트 변경 기록을 확인합니다.

| 표시 항목 | 설명 |
|---|---|
| 변경 전/후 벨트 | 어떤 벨트에서 어떤 벨트로 변경되었는지 |
| 승급일 | 벨트가 변경된 날짜 |
| 승급자 | 누가 승급을 실행했는지 |
| 메모 | 승급 시 기록된 메모 |

## 벨트 일괄 관리

여러 회원의 벨트를 한 번에 변경하려면 [승급식](../ceremonies.md) 기능을 이용하세요. 출석 기준으로 후보자를 자동 추출하고, 검토 후 일괄 승급을 실행합니다.
