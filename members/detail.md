# 회원 상세

회원 목록에서 회원을 클릭하면 **상세 프로필** 페이지로 이동합니다.\
한 화면에서 해당 회원의 모든 정보를 확인하고 관리할 수 있습니다.

---

## 프로필 정보

| 항목 | 설명 |
|---|---|
| 이름 | 회원 실명 |
| 벨트 | 현재 벨트 색상 ([벨트 관리](belt-management.md)에서 변경) |
| 역할 | 지도자(PROFESSOR) / 정회원(MEMBERSHIP) / 게스트(GUEST) |
| 운동 시작일 | 수동 입력 가능 |
| 건강 메모 | 부상 이력, 주의 사항 등 |
| 비상 연락처 | 응급 상황 시 연락처 |

{% hint style="info" %}
운동 시작일, 건강 메모, 비상 연락처는 회원 상세 페이지에서 직접 수정할 수 있습니다.
{% endhint %}

## 수업권

### 현재 수업권

현재 활성 수업권 정보를 표시합니다.

- **기간제**: 시작일, 종료일, 잔여일
- **횟수제**: 총 횟수, 잔여 횟수

### 구매 이력

해당 회원의 전체 수업권 구매/갱신 이력을 조회합니다.

### 수업권 갱신

{% hint style="warning" %}
수업권 갱신은 [수업권](../membership-plans.md)이 1개 이상 등록되어 있어야 합니다.
{% endhint %}

- 기존 수업권 만료 전/후 새 수업권을 등록합니다.
- **추천 시작일**이 자동으로 계산됩니다 (기존 수업권 종료일 다음 날).
- 갱신 전 **미리보기**로 적용 결과를 확인할 수 있습니다.

### 환불 처리

수업권을 환불합니다. 사용 기간/횟수에 따른 사용 비율이 자동으로 계산됩니다.

### 횟수 조정

횟수제 수업권의 잔여 횟수를 수동으로 조정합니다. 조정 사유가 기록됩니다.

## 계약서

해당 회원의 서명된 [계약서](../contracts/) 목록을 확인합니다.

- **서명 완료** 상태의 계약서는 결제 확인 처리를 할 수 있습니다.
- 계약서를 클릭하면 PDF 미리보기가 가능합니다.

## 메시지 이력

해당 회원에게 발송된 [공지](../announcements/manual.md), [자동 메시지](../announcements/auto-messages.md) 이력을 확인합니다.

## 메모

지도자가 작성하는 **내부 메모**입니다. 회원에게는 비공개됩니다.

- 건강 상태, 부상 이력, 특이 사항 등을 기록합니다.
- 최대 200자까지 입력 가능합니다.
- 여러 개의 메모를 작성할 수 있습니다.
