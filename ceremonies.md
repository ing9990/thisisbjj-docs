# 승급식

벨트 승급 행사를 체계적으로 관리합니다.\
출석 기준으로 후보자를 자동 추출하고, 검토 후 일괄 승급을 실행합니다.

---

{% hint style="warning" %}
승급식 기능은 [지도자 인증](getting-started/leader-verification.md)과 [PARTNER 구독](subscription.md)이 모두 필요합니다.
{% endhint %}

## 승급식 워크플로우

```
① 승급식 생성 → ② 후보자 검색 → ③ 후보자 확정 → ④ 승급 실행
```

### ① 승급식 생성

[주짓주짓 매니저](https://manager.thisisbjj.com) → **승급식** 메뉴에서 새 승급식을 만듭니다.

| 항목 | 설명 |
|---|---|
| 승급식 이름 | 예: "2024년 3월 정기 승급식" |
| 날짜 | 승급식 예정일 |
| 최소 출석 기준(일) | 후보자 자동 추출 시 기준 |

{% hint style="info" %}
이전 승급식의 출석 기준이 자동으로 불러와집니다. 매번 다시 입력할 필요가 없습니다.
{% endhint %}

### ② 후보자 자동 검색

**"후보자 검색"** 버튼을 클릭하면 조건에 맞는 회원이 자동으로 추출됩니다.

**추출 조건**
- 정회원(MEMBERSHIP) 또는 지도자(PROFESSOR) 역할
- 최고 벨트에 도달하지 않은 회원
- 설정한 최소 출석 기준 충족

### ③ 후보자 관리

추출된 후보자를 검토하고 필요에 따라 조정합니다.

| 기능 | 설명 |
|---|---|
| **승급 벨트 변경** | 자동 추천 벨트 대신 다른 벨트로 직접 지정 |
| **후보자 제외** | 이번 승급식에서 특정 회원 제외 |
| **초기 벨트 설정** | 타 도장 이적 회원의 현재 벨트 지정 |

**"확정"** 버튼을 클릭하면 후보자 목록이 잠깁니다.

{% hint style="info" %}
확정 후에도 **"확정 취소"** 로 되돌릴 수 있습니다. 실행 전까지는 수정이 가능합니다.
{% endhint %}

### ④ 승급 실행

**"승급 실행"** 버튼을 클릭하면:

1. 모든 후보자의 벨트가 **일괄 업데이트**됩니다.
2. 각 회원의 [승급 이력](members/belt-management.md)에 기록됩니다.
3. 승급자, 승급일, 승급식 정보가 함께 기록됩니다.

{% hint style="danger" %}
승급 실행 후에는 **되돌릴 수 없습니다**. 실행 전에 후보자 목록을 충분히 검토하세요.
{% endhint %}

## 개별 벨트 관리

승급식 없이 개별 회원의 벨트를 변경하려면 [벨트 관리](members/belt-management.md) 페이지를 이용하세요.
