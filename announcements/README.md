# 자동 메시지

조건에 따라 회원에게 **자동으로 알림톡**이 발송됩니다.

---

{% hint style="warning" %}
자동 메시지 기능은 [지도자 인증](../getting-started/leader-verification.md)이 완료되어야 사용할 수 있습니다.
{% endhint %}

## 자동 메시지 유형

| 유형 | 설명 |
|---|---|
| **회원권 만료 알림** | 만료 N일 전 자동 알림 |
| **장기 미출석 안내** | N일간 출석이 없는 회원에게 안내 |
| **신규 회원 환영** | 가입 즉시 환영 메시지 발송 |
| **잔여 횟수 알림** | 횟수제 회원권 잔여 N회 이하 시 알림 |
| **미결제 알림** | 계약서 서명 후 결제가 안 된 경우 안내 |
