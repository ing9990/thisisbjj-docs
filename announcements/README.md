# 공지 & 자동 메시지

회원들에게 단체 공지를 보내거나, 조건에 따라 자동으로 메시지를 발송합니다.

---

{% hint style="warning" %}
공지 & 자동 메시지 기능은 [지도자 인증](../getting-started/leader-verification.md)과 [PARTNER 구독](../subscription.md)이 모두 필요합니다.
{% endhint %}

## 기능 구성

| 기능 | 설명 | 바로가기 |
|---|---|---|
| 공지 발송 | 수동으로 단체 메시지 전송 | [공지 발송](manual.md) |
| 발송 이력 | 보낸 공지의 전달/열람 상태 확인 | [공지 발송](manual.md) |
| 자동 메시지 | 조건 충족 시 자동 발송 규칙 설정 | [자동 메시지](auto-messages.md) |

## 발송 채널

| 채널 | 설명 |
|---|---|
| **카카오 알림톡** | 카카오톡으로 발송 (알림톡 수신 동의 불필요) |
| **SMS** | 문자 메시지로 발송 |

---

## 상세 기능

- [공지 발송](manual.md) — 수동 단체 공지 보내기
- [자동 메시지](auto-messages.md) — 조건부 자동 알림 설정
