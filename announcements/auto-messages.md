# 자동 메시지

미리 설정해 두면 조건에 맞는 회원에게 **자동으로 메시지**가 발송됩니다.

---

{% hint style="warning" %}
자동 메시지는 [지도자 인증](../getting-started/leader-verification.md)과 [PARTNER 구독](../subscription.md)이 모두 필요합니다.
{% endhint %}

## 자동 메시지 유형

[주짓주짓 매니저](https://manager.thisisbjj.com) → **공지** → **자동 메시지** 메뉴에서 설정합니다.

| 유형 | 설명 | 설정값 |
|---|---|---|
| **수업권 만료 알림** | 만료 N일 전 자동 알림 | 7일, 14일, 30일 |
| **신규 회원 환영** | 가입 즉시 환영 메시지 발송 | — |
| **장기 미출석 안내** | N일간 출석이 없는 회원에게 안내 | N일 설정 |
| **잔여 횟수 알림** | 횟수제 수업권 잔여 N회 이하 시 알림 | N회 설정 |
| **미결제 알림** | [계약서](../contracts/) 서명 후 결제가 안 된 경우 안내 | — |

## 설정 방법

1. 원하는 자동 메시지 유형의 **활성/비활성** 토글을 전환합니다.
2. 설정값이 있는 유형은 기준(N일, N회)을 도장 상황에 맞게 조정합니다.
3. 설정은 즉시 적용됩니다.

{% hint style="info" %}
자동 메시지는 조건 충족 시 **카카오 알림톡**으로 발송됩니다.
{% endhint %}

## 활용 예시

| 상황 | 추천 설정 |
|---|---|
| 회원 이탈 방지 | 만료 알림 7일 + 14일 전, 미출석 14일 |
| 신규 회원 케어 | 환영 메시지 활성화 |
| 횟수제 운영 | 잔여 횟수 3회 이하 알림 |
| 미수금 관리 | 미결제 알림 활성화 |

## 발송 이력 확인

자동 메시지 발송 이력은 [회원 상세](../members/detail.md) 페이지의 **메시지 이력**에서 확인할 수 있습니다.
