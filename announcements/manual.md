# 공지 발송

회원들에게 단체 메시지를 보냅니다.

---

## 사전 조건

{% hint style="danger" %}
공지 발송을 위해 아래 항목이 **모두** 완료되어야 합니다.

1. [전화번호 인증](../getting-started/phone-verification.md)
2. [지도자 인증](../getting-started/leader-verification.md)

## 공지 작성

[주짓주짓 매니저](https://manager.thisisbjj.com) → **공지** → **공지 발송** 메뉴에서 진행합니다.

### 내용 작성

| 항목 | 설명 |
|---|---|
| 제목 | 공지 제목 |
| 본문 | 공지 내용 |
| 발송 채널 | 카카오 알림톡 / SMS 선택 |

### 수신자 필터

전체 회원에게 보내거나, 조건에 맞는 회원에게만 보낼 수 있습니다.

| 필터 | 설명 |
|---|---|
| 벨트 색상 | 특정 벨트 회원만 |
| 연령대 | 특정 연령대만 |
| 만료 임박 | N일 이내 회원권 만료 예정 |
| 가입일 범위 | 특정 기간에 가입한 회원 |
| 회원권 유형 | 기간제 / 횟수제 |

{% hint style="info" %}
**활용 예시**: 흰 벨트 회원에게만 기초 특강 안내, 만료 임박 회원에게 갱신 할인 안내 등
{% endhint %}

## 발송 이력

**공지** → **발송 이력** 메뉴에서 보낸 공지의 상태를 확인합니다.

- 발송 성공/실패 건수
- 열람율
- 실패 사유 (번호 오류, 수신 거부 등)

{% hint style="info" %}
이전에 보낸 공지를 **재사용**할 수 있습니다. 발송 이력에서 **"다시 보내기"** 를 클릭하면 내용이 자동으로 채워집니다.
{% endhint %}
