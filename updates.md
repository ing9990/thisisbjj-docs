# Updates

주짓주짓 매니저에 추가된 기능과 개선 사항을 시간 역순으로 안내합니다.

---

## 2026년 3월 27일

### 결제 상세 페이지

결제 건별 상세 정보를 타임라인 형태로 확인할 수 있습니다. 결제일, 환불일, 계약서 정보가 시간순으로 표시됩니다.

* 결제 상세에서 **계약서 PDF 다운로드** 가능
* 환불된 회원권은 결제 행과 환불 행이 **함께 표시**되어 이력 추적이 쉬워졌습니다

### 정산 결제 내역 개선

* 결제 내역 테이블에 **컬럼 정렬** 기능 추가
* **Excel 내보내기**에 환불 데이터 포함
* 결제 금액 표시 가독성 향상 (천 단위 콤마)

### 만료 알림 버튼

회원권 만료가 임박한 회원에게 **만료 알림 메시지**를 보낼 수 있는 버튼이 추가되었습니다.

### 프로필 설정 페이지

도장 관리자가 **프로필 설정** 페이지에서 벨트를 선택할 수 있는 모달이 추가되었습니다.

### 키오스크 출석 시 벨트 자동 설정

키오스크 출석 체크 시 **벨트 정보가 자동으로 활성화**됩니다. 별도 설정 없이 출석과 동시에 벨트가 반영됩니다.

## 2026년 3월 26일

### 계약서 서명 실시간 알림 (SSE)

회원이 계약서에 서명하면 **브라우저 알림**이 실시간으로 표시됩니다. 계약서 발송 페이지를 열어놓지 않아도 알림을 받을 수 있습니다.

* 서명 완료 시 브라우저 푸시 알림
* 계약서 발송 페이지에서 실시간 상태 업데이트
* 서명된 계약서로 바로 이동하는 링크 제공

### UI 현대화

* **정산 대시보드** 및 결제 관리 UI 전면 리디자인
* **도장 정보** 관리 페이지 디자인 개선
* **서명된 계약서** 페이지 리디자인

### 대시보드 활동 피드 강화

대시보드의 최근 활동 피드가 **여러 소스의 이벤트를 통합**하여 보여줍니다. 출석, 계약서 서명, 회원 가입 등 도장에서 일어나는 모든 활동을 한눈에 확인하세요.

### 승급식 후보 관리 개선

승급식 후보자 목록 관리가 개선되었습니다.

### 스케줄러 Slack 알림

10분 간격으로 실행되는 배치 작업(만료 알림, 결제 등)의 **결과 요약이 Slack으로 전송**됩니다.

## 2026년 3월 24~25일

### 횟수 기반 회원권

기간 기반 회원권 외에 **횟수 기반 회원권**을 생성할 수 있습니다.

* 회원권 유형 토글 (기간/횟수)
* 횟수 기반 회원권의 출석 시 자동 차감
* 잔여 횟수 부족 회원 대시보드 알림
* 계약서 서명/PDF에 회원권 유형 자동 반영

### 회원 목록 개선

* **AJAX 테이블** 레이아웃으로 전면 전환
* 잔여일, 최근 출석, 수련 기간 기준 **정렬** 기능
* 벨트/나이/키워드 **필터** 강화

### 키오스크 성공 화면 개선

출석 체크 성공 시 **링 프로그레스** 애니메이션으로 변경되었습니다. 파란색은 기간 기반, 금색은 횟수 기반 회원권을 나타냅니다.

### 매니저 랜딩 페이지 리디자인

기능 쇼케이스 레이아웃으로 매니저 랜딩 페이지가 리디자인되었습니다.

## 2026년 3월 23일

### 환불 기능

회원 상세의 구매 이력에서 **환불 버튼**을 눌러 환불을 처리할 수 있습니다.

* 환불 전 **확인 화면**에서 환불 예상 금액 표시
* 환불 완료 시 회원 역할 자동 동기화
* 환불 내역이 정산 데이터에 반영

### 온보딩 체크리스트

처음 매니저에 접속한 도장에게 **셋업 체크리스트**가 표시됩니다. 도장 정보, 회원권 생성, 계약서 설정 등 필수 단계를 안내합니다.

### 만료 회원권 출석 차단

회원권이 만료된 회원은 **키오스크에서 출석 체크가 차단**됩니다.

* 전화번호 자동 정규화, 날짜 자동 파싱

### 이용약관 및 환불 정책

서비스 이용약관과 환불 정책 페이지가 추가되었습니다.

## 2026년 3월 19일

### 자동 메시지

특정 이벤트 발생 시 **자동으로 메시지를 보내는 규칙**을 설정할 수 있습니다.

* **환영 메시지**: 새 회원 가입 시 자동 발송
* **승급 축하**: 벨트 승급 시 자동 발송
* 규칙별 발송 채널 설정 (카카오 알림톡, 앱 푸시)
* 발송 이력 통합 조회 페이지

### 회원 상세 메시지 이력

회원 상세 페이지에서 **해당 회원에게 보낸 메시지 이력**을 확인할 수 있습니다.

### KPI 툴팁

대시보드 KPI 숫자 위에 마우스를 올리면 **계산 방법 상세**가 표시됩니다.

## 2026년 3월 16~17일

### 사업자 인증 페이지

Toss 스타일의 **4단계 위저드**로 사업자 인증 과정이 개편되었습니다.

1. 인증 방법 선택
2. 전화번호 인증 (OTP)
3. 사업자등록증 업로드
4. 인증 완료 대기

### 도장 정보 인라인 수정

도장 정보 페이지에서 **각 필드를 클릭하여 바로 수정**할 수 있습니다. 저장 버튼 없이 **자동 저장**됩니다.

### 벨트 체계 확장

기존 성인 벨트 외에 **유소년(Youth)** 및 **마스터(Master)** 벨트가 추가되어 총 **12색**으로 확장되었습니다.

* 대시보드 벨트 분포 차트 12색 지원
* 공지 발송 벨트 필터 12색 지원
* 3탭 UI (유소년/성인/마스터) 벨트 승급 모달

* 오늘 출석 현황 페이지

### 회원 상세 페이지 리디자인
