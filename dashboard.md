# 대시보드

[주짓주짓 매니저](https://manager.thisisbjj.com)에 로그인하면 가장 먼저 보이는 화면입니다.\
도장의 현황을 한눈에 파악할 수 있습니다.

---

{% hint style="warning" %}
대시보드를 사용하려면 [지도자 인증](getting-started/leader-verification.md)이 완료되어야 합니다.
{% endhint %}

## 핵심 지표 (KPI 카드)

| 지표 | 설명 |
|---|---|
| **활성 회원 수** | 현재 회원권이 유효한 회원 |
| **이번 달 신규 회원** | 이번 달 새로 가입한 회원 수 |
| **이번 달 매출** | 이번 달 결제 확인된 금액 합계 |
| **이탈 회원** | 회원권 만료 후 갱신하지 않은 회원 |

## 벨트 분포 차트

도장 회원들의 벨트 색상별 분포를 시각적으로 확인합니다.

## 최근 활동

신규 가입, 출석, 결제 등 도장의 최근 활동 내역이 타임라인으로 표시됩니다.

## 출석 시 벨트 설정 옵션

도장 설정 카드에서 **"출석체크로 벨트설정하기"** 토글을 켤 수 있습니다.\
활성화하면 벨트가 미설정된 회원이 [키오스크](attendance/kiosk.md)에서 체크인할 때 벨트 색상 선택 화면이 1회 나타납니다.

## PARTNER 전용 기능

아래 항목은 [PARTNER 구독](subscription.md) 시 추가로 제공됩니다.

| 기능 | 설명 |
|---|---|
| 회원 추이 (6개월) | 월별 누적 회원 수 변화 그래프 |
| 출석률 추이 (6개월) | 월별 평균 출석률 그래프 |
| 매출 추이 (6개월) | 월별 매출 변화 그래프 |
| 만료 임박 회원 | 10일 이내 회원권 만료 예정 회원 목록 (최대 8명) |
| 오늘의 출석 현황 | 오늘 키오스크/앱으로 체크인한 회원 스냅샷 |
| 전월 비교 | 매출, 신규 회원, 이탈 수치를 전월과 비교 |
