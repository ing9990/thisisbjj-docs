# 데이터 이관

기존 도장 관리 시스템(다짐 등)에서 회원 데이터를 가져옵니다.

---

## 사전 조건

{% hint style="danger" %}
데이터 이관을 위해 아래 항목이 **모두** 완료되어야 합니다.

1. [지도자 인증](getting-started/leader-verification.md)
2. [PARTNER 구독](subscription.md) (FREE는 15명 제한)
{% endhint %}

## 이관 과정

```
① 기존 시스템에서 Excel 다운로드
    ↓
② 매니저에서 Excel 업로드
    ↓
③ 데이터 검증 & 미리보기
    ↓
④ 관리자 승인 후 이관 완료
```

### ① Excel 파일 준비

기존 도장 관리 시스템에서 회원 데이터를 **Excel 파일**로 내보냅니다.

| 항목 | 설명 |
|---|---|
| 지원 형식 | `.xlsx`, `.xls` |
| 최대 크기 | 10MB |
| 지원 시스템 | 다짐 (DaGym) 형식 |

### ② 파일 업로드

[주짓주짓 매니저](https://manager.thisisbjj.com) → **데이터 이관** 메뉴에서 파일을 업로드합니다.

- 업로드 시 자동으로 컬럼 매핑 및 데이터 검증이 진행됩니다.
- 오류가 발견되면 어떤 항목에 문제가 있는지 안내됩니다.

### ③ 미리보기 & 확인

이관될 회원 데이터를 미리 확인합니다.

### ④ 관리자 승인

데이터 검증이 완료되면 관리자가 최종 승인합니다.

{% hint style="info" %}
FREE 요금제의 경우 총 회원 수가 **15명을 초과**하면 업로드가 제한됩니다. [PARTNER 구독](subscription.md)으로 전환 후 진행해 주세요.
{% endhint %}

## 이관 상태 확인

업로드 후 **데이터 이관** 페이지에서 진행 상황을 확인할 수 있습니다.

| 상태 | 설명 |
|---|---|
| **대기 중** | 관리자 검토 대기 |
| **미리보기** | 데이터 확인 진행 중 |
| **완료** | 이관 성공 |
| **거절** | 데이터 오류로 거절 (수정 후 재업로드 필요) |
| **롤백** | 이관이 취소됨 |
