# 회원 서명 과정

회원이 계약서 링크를 받은 후 서명을 완료하기까지의 과정입니다.\
이 페이지는 **지도자가 회원에게 안내할 때** 참고용으로 활용하세요.

---

## 서명 과정

회원은 별도 앱 설치 없이 **모바일 브라우저**에서 바로 서명합니다.

```
① 링크 클릭 → ② 개인정보 입력 → ③ 계약 내용 확인 → ④ 서명 → ⑤ 완료
```

### ① 링크 클릭

카카오 알림톡 또는 SMS로 받은 링크를 클릭합니다.

### ② 개인정보 입력

| 입력 항목 | 설명 |
|---|---|
| 이름 | 본인 실명 |
| 주민등록번호 | 본인 확인용 (암호화 저장) |
| 연락처 | 휴대폰 번호 |
| 보호자 정보 | 미성년자인 경우 보호자 이름, 연락처 |

### ③ 계약 내용 확인

회원권 정보, 환불 규정, 동의 항목 등 계약서 내용을 확인합니다.

### ④ 서명

화면에 직접 **손가락 또는 마우스**로 서명합니다.

### ⑤ 완료

서명이 완료되면 **계약서 PDF가 자동 생성**됩니다.\
지도자는 [서명된 계약서 관리](signed-management.md)에서 확인할 수 있습니다.

---

{% hint style="info" %}
서명 링크는 **2시간** 동안 유효합니다. 시간이 초과된 경우 [계약서 발송](sending.md)에서 다시 보내주세요.
{% endhint %}
