# 서명된 계약서 관리

회원이 서명을 완료한 계약서를 조회하고 결제 상태를 관리합니다.

---

## 계약서 상태

| 상태 | 의미 | 지도자 액션 |
|---|---|---|
| 🟡 **서명 완료** | 회원이 서명했지만 결제가 확인되지 않은 상태 | 결제 확인 처리 |
| 🟢 **결제 확인** | 지도자가 결제를 확인한 상태 | — |

## 계약서 목록

[주짓주짓 매니저](https://manager.thisisbjj.com) → **계약서** → **서명된 계약서** 메뉴에서 전체 목록을 확인합니다.

- 회원 이름으로 검색할 수 있습니다.
- 결제 상태(서명 완료 / 결제 확인)별로 필터링할 수 있습니다.

## 결제 확인

서명 완료 상태의 계약서에서 **"결제 확인"** 버튼을 클릭하면:

1. 계약서 상태가 **결제 확인**으로 변경됩니다.
2. 해당 회원권의 **이용 기간이 자동으로 시작**됩니다.
3. [회원 상세](../members/detail.md)에서 활성 회원권으로 표시됩니다.

{% hint style="info" %}
[회원 상세](../members/detail.md) 페이지에서도 계약서별 결제 확인이 가능합니다.
{% endhint %}

## PDF 미리보기

계약서를 클릭하면 서명된 **PDF 미리보기**를 볼 수 있습니다.\
지도자 서명과 회원 서명이 함께 표시됩니다.

## 계약서 검증

서명된 계약서마다 고유한 검증 URL이 생성됩니다.\
**비밀번호 보호** 열람이 가능하므로, 분쟁 발생 시 원본 확인 용도로 활용할 수 있습니다.
