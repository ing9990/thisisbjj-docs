# 계약서 관리

종이 계약서 없이 **모바일 전자 계약서**로 회원 등록을 진행합니다.\
회원은 별도 앱 설치 없이 모바일 브라우저에서 바로 서명할 수 있습니다.

---

{% hint style="warning" %}
계약서 기능은 [지도자 인증](../getting-started/leader-verification.md)이 완료되어야 사용할 수 있습니다.\
계약서 **발송**은 [PARTNER 구독](../subscription.md)이 필요합니다.
{% endhint %}

## 전체 흐름

```
계약서 템플릿 만들기 → 수업권과 연결 → 회원에게 발송 → 회원 서명 → 결제 확인
```

| 단계 | 설명 | 바로가기 |
|---|---|---|
| ① 템플릿 만들기 | 도장에 맞는 계약서 양식 작성 | 이 페이지 아래 |
| ② 수업권 연결 | 템플릿을 [수업권](../membership-plans.md)에 연결 | [수업권 관리](../membership-plans.md) |
| ③ 발송 | 회원에게 카카오/SMS로 링크 전송 | [계약서 발송](sending.md) |
| ④ 회원 서명 | 회원이 모바일에서 서명 | [회원 서명 과정](signing-flow.md) |
| ⑤ 관리 | 서명된 계약서 조회/결제 확인 | [서명된 계약서 관리](signed-management.md) |

---

## 계약서 템플릿

[주짓주짓 매니저](https://manager.thisisbjj.com) → **계약서** 메뉴에서 템플릿을 관리합니다.

### 기본 제공 항목

템플릿 생성 시 아래 항목이 기본으로 포함됩니다. 필요에 따라 수정하거나 삭제할 수 있습니다.

- **환불 규정** — 체육시설법 기준 환불 정책
- **사진/영상 촬영 동의** — 수업 중 촬영 및 홍보 활용 동의
- **부상 면책 조항** — 운동 중 부상에 대한 면책
- **자유 입력 항목** — 도장별 커스텀 조항 추가

### 템플릿 편집

섹션 단위로 내용을 추가, 수정, 삭제할 수 있습니다.\
드래그로 섹션 순서를 변경할 수 있습니다.

### 지도자 서명 등록

계약서에 표시될 **지도자 서명 이미지**를 미리 등록합니다.\
회원이 서명하면 지도자 서명과 함께 PDF가 생성됩니다.

**계약서** → **지도자 서명** 메뉴에서 서명 이미지를 업로드하세요.

{% hint style="danger" %}
연결된 [수업권](../membership-plans.md)이 있는 템플릿은 삭제할 수 없습니다.
{% endhint %}
