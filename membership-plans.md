# 회원권 관리

도장에서 판매할 회원권(요금제)을 만들고 관리합니다.

---

{% hint style="warning" %}
회원권을 생성하려면 [지도자 인증](getting-started/leader-verification.md)이 완료되어야 합니다.
{% endhint %}

## 회원권 유형

| 유형 | 설명 | 예시 |
|---|---|---|
| **기간제** (PERIOD) | 등록일부터 N일간 무제한 수업 | 1개월권(30일), 3개월권(90일), 6개월권(180일) |
| **횟수제** (COUNT) | N회 수업 이용권 | 10회권, 20회권 |

## 회원권 만들기

[주짓주짓 매니저](https://manager.thisisbjj.com) → **회원권** 메뉴에서 새 회원권을 만듭니다.

| 항목 | 설명 | 필수 |
|---|---|---|
| 회원권 이름 | 예: "주짓수 3개월 정기권" | ✅ |
| 유형 | 기간제 또는 횟수제 | ✅ |
| 기간(일) / 횟수 | 기간제: 30, 90, 180 등 / 횟수제: 10, 20 등 | ✅ |
| 가격 | 정상 판매 가격 | ✅ |
| 원래 가격 | 할인 전 가격 (할인 표시용, 선택) | |
| 대표 이미지 | 회원권 카드에 표시될 이미지 (템플릿 제공) | |
| 계약서 템플릿 | 이 회원권과 연결할 [계약서 양식](contracts/) | |

{% hint style="info" %}
**복제 기능**: 기존 회원권을 기반으로 새 회원권을 빠르게 만들 수 있습니다. 회원권 목록에서 복제할 회원권을 선택하세요.
{% endhint %}

## 회원권 목록 관리

- **드래그**로 표시 순서를 변경할 수 있습니다.
- 회원권을 수정/삭제할 수 있습니다.

{% hint style="danger" %}
활성 계약이 연결된 회원권은 **삭제할 수 없습니다**. 먼저 연결된 계약을 처리한 후 삭제하세요.
{% endhint %}

## 계약서 연결

회원권에 [계약서 템플릿](contracts/)을 연결하면, [계약서 발송](contracts/sending.md) 시 해당 회원권과 계약서가 자동으로 매칭됩니다.
