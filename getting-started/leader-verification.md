# 지도자 인증 (사업자 인증)

도장의 소유권을 확인하는 과정입니다. 지도자 인증이 완료되면 [주짓주짓 매니저](https://manager.thisisbjj.com)의 모든 기능을 사용할 수 있습니다.

---

## 사전 조건

{% hint style="warning" %}
지도자 인증 전에 [전화번호 인증](phone-verification.md)을 먼저 완료해 주세요.
{% endhint %}

## 인증 과정

### STEP 1 — 도장 등록

두 가지 방법 중 선택합니다.

| 방법 | 설명 |
|---|---|
| **새 도장 만들기** | 카카오 장소 검색으로 도장을 신규 등록합니다. |
| **기존 도장 연결** | 이미 주짓주짓에 등록된 도장에 소유권 인증 요청을 보냅니다. |

### STEP 2 — 사업자 인증

아래 정보를 제출합니다.

- **사업자등록증** (이미지 업로드)
- **대표자 연락처**

### STEP 3 — 승인 대기

- 제출된 정보를 관리자가 검토합니다.
- 보통 **1~2 영업일** 내에 승인이 완료됩니다.
- 승인 결과는 **카카오 알림톡**으로 안내됩니다.

### STEP 4 — 완료

승인이 완료되면 도장의 **지도자(PROFESSOR)** 권한이 부여됩니다.
