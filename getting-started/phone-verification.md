# 전화번호 인증

본인 확인을 위해 휴대폰 번호를 인증합니다. 인증된 전화번호는 계약서 자동 연결, 회원 식별 등에 사용됩니다.

---

## 인증 과정

1. 로그인 후 전화번호 입력 화면이 나타납니다.
2. 휴대폰 번호를 입력하고 인증번호를 요청합니다.
3. SMS로 받은 인증번호를 입력하면 완료됩니다.

{% hint style="warning" %}
전화번호 인증은 **1회만** 진행하면 됩니다. 이후에는 자동으로 인증 상태가 유지됩니다.
{% endhint %}
