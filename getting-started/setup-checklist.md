# 셋업 체크리스트

[지도자 인증](leader-verification.md) 완료 후 [대시보드](../dashboard.md)에 **셋업 체크리스트**가 표시됩니다.\
체크리스트를 따라가면 [주짓주짓 매니저](https://manager.thisisbjj.com)의 주요 설정을 빠짐없이 완료할 수 있습니다.

---

## 체크리스트 항목

| 순서 | 항목 | 설명 |
|---|---|---|
| ① | [도장 정보 입력](../gym-info.md) | 로고, 연락처, 소개글 |
| ② | 첫 회원 등록 | [회원 관리](../members/)에서 확인 |
| ③ | [회원권 생성](../membership-plans.md) | 기간제/횟수제 회원권 1개 이상 |
| ④ | [계약서 템플릿 설정](../contracts/) | 계약서 양식 만들기 |
| ⑤ | [키오스크 세팅](../attendance/kiosk.md) | 출석 체크인 준비 |
| ⑥ | [첫 계약서 발송](../contracts/sending.md) | 회원에게 전자 계약서 보내기 |
| ⑦ | [데이터 이관](../data-migration.md) | 기존 시스템 데이터 가져오기 |
| ⑧ | [승급식 생성](../ceremonies.md) | 벨트 승급 행사 예약 |


## 체크리스트 관리

- 각 항목을 완료하면 자동으로 체크 표시됩니다.
- 일부 항목(키오스크 세팅, 승급식 등)은 **수동 체크**가 필요합니다.
- 모든 항목을 완료하지 않아도 매니저 사용에 지장은 없습니다.
- 체크리스트는 언제든 **숨기거나 다시 표시**할 수 있습니다.
