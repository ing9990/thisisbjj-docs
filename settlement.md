# 정산

도장의 매출과 결제 내역을 관리합니다.

---

{% hint style="warning" %}
정산 기능은 [지도자 인증](getting-started/leader-verification.md)이 완료되어야 사용할 수 있습니다.
{% endhint %}

## 정산 대시보드

[주짓주짓 매니저](https://manager.thisisbjj.com) → **정산** 메뉴에서 매출 현황을 확인합니다.

| 지표 | 설명 |
|---|---|
| **총 매출** | 조회 기간 내 전체 결제 금액 |
| **신규 회원 매출** | 신규 회원의 결제 금액 및 건수 |
| **갱신 매출** | 기존 회원 재등록 금액 |
| **결제 수단별 비중** | 카드 / 현금 / 이체 비율 |

## 결제 내역

모든 결제 건을 날짜순으로 조회합니다.

| 표시 항목 | 설명 |
|---|---|
| 결제일 | 결제가 확인된 날짜 |
| 회원명 | 결제한 회원 |
| 회원권명 | 결제한 회원권 |
| 금액 | 결제 금액 |
| 결제 수단 | 카드, 현금, 이체 등 |

{% hint style="info" %}
결제 수단이 잘못 기록된 경우 **수동으로 수정**할 수 있습니다. 결제 내역에서 해당 건을 클릭하세요.
{% endhint %}

## 리포트 내보내기

원하는 기간을 설정하여 **Excel(XLSX) 파일**로 다운로드합니다.

- 세무/회계 목적으로 활용할 수 있습니다.
- 결제일, 회원명, 회원권명, 금액, 결제 수단 등 모든 항목이 포함됩니다.

---

## 미수금 관리

[계약서](contracts/) 서명 후 결제가 확인되지 않은 건을 추적할 수 있습니다.\
[자동 메시지](announcements/auto-messages.md)의 **미결제 알림**을 활성화하면 자동으로 안내됩니다.

---

## 수납 상세

결제 내역에서 개별 건을 클릭하면 **수납 상세** 화면으로 이동합니다. 해당 결제의 전체 흐름과 상세 정보를 한눈에 확인할 수 있습니다.

<figure><img src=".gitbook/assets/수납 상세.png" alt="수납 상세 화면"><figcaption>수납 상세 화면</figcaption></figure>

### 타임라인

결제 건의 주요 이벤트를 시간순으로 표시합니다.

| 이벤트 | 설명 |
|---|---|
| **계약 서명** | 회원이 계약서에 서명한 시점. 서명한 회원권명이 함께 표시됩니다. |
| **결제 완료** | 결제가 확인된 시점. 결제 수단과 금액이 표시됩니다. |
| **이용 시작** | 회원권 이용이 시작된 날짜입니다. |
| **이용 종료** | 회원권 이용 기간이 만료된 날짜입니다. |
| **환불 처리** | 환불이 진행된 경우 환불 금액과 처리자가 표시됩니다. |

{% hint style="info" %}
타임라인은 해당 결제 건에 발생한 이벤트만 표시됩니다. 환불이 없는 정상 결제 건에는 환불 처리 항목이 나타나지 않습니다.
{% endhint %}

### 결제 정보

| 항목 | 설명 |
|---|---|
| 회원명 | 결제한 회원 이름 |
| 플랜 | 결제한 회원권명 (예: 3개월 회원권) |
| 타입 | 회원권 유형 (기간제 / 횟수제) |
| 금액 | 결제 금액 |
| 결제 수단 | 카드, 현금, 이체 등 |
| 결제일 | 결제가 확인된 날짜 |
| 상태 | 결제 상태 (정상, 취소, 환불됨 등) |

### 이용 정보

| 항목 | 설명 |
|---|---|
| 시작일 | 회원권 이용 시작 날짜 |
| 종료일 | 회원권 이용 종료 날짜 |
| 총 기간 | 전체 이용 기간 (일 단위) |
| 잔여 | 남은 이용 기간 (일 단위) |
| 출석 | 이용 기간 중 출석 횟수 |
| 이용률 | 출석 횟수 기반 이용률 (프로그레스 바로 시각화) |

### 환불 정보

환불이 발생한 결제 건에만 표시됩니다.

| 항목 | 설명 |
|---|---|
| 환불 금액 | 환불된 금액 |
| 환불일 | 환불 처리 날짜 |
| 환불 사유 | 환불 사유 (입력된 경우) |
| 처리자 | 환불을 처리한 관리자 |

### 연결 정보

해당 결제 건과 관련된 문서 및 회원 정보로 바로 이동할 수 있습니다.

| 항목 | 설명 |
|---|---|
| 계약서 | 결제 당시 서명된 계약서 상태 표시. **계약서 PDF 다운로드** 버튼으로 원본 계약서를 내려받을 수 있습니다. |
| 회원 상세 | **회원 정보 보기** 링크를 클릭하면 해당 [회원 상세](members/detail.md) 페이지로 이동합니다. |
