# 정산

도장의 매출과 결제 내역을 관리합니다.

---

{% hint style="warning" %}
정산 기능은 [지도자 인증](getting-started/leader-verification.md)과 [PARTNER 구독](subscription.md)이 모두 필요합니다.
{% endhint %}

## 정산 대시보드

[주짓주짓 매니저](https://manager.thisisbjj.com) → **정산** 메뉴에서 매출 현황을 확인합니다.

| 지표 | 설명 |
|---|---|
| **총 매출** | 조회 기간 내 전체 결제 금액 |
| **신규 회원 매출** | 신규 회원의 결제 금액 및 건수 |
| **갱신 매출** | 기존 회원 재등록 금액 |
| **결제 수단별 비중** | 카드 / 현금 / 이체 비율 |

## 결제 내역

모든 결제 건을 날짜순으로 조회합니다.

| 표시 항목 | 설명 |
|---|---|
| 결제일 | 결제가 확인된 날짜 |
| 회원명 | 결제한 회원 |
| 회원권명 | 결제한 회원권 |
| 금액 | 결제 금액 |
| 결제 수단 | 카드, 현금, 이체 등 |

{% hint style="info" %}
결제 수단이 잘못 기록된 경우 **수동으로 수정**할 수 있습니다. 결제 내역에서 해당 건을 클릭하세요.
{% endhint %}

## 리포트 내보내기

원하는 기간을 설정하여 **Excel(XLSX) 파일**로 다운로드합니다.

- 세무/회계 목적으로 활용할 수 있습니다.
- 결제일, 회원명, 회원권명, 금액, 결제 수단 등 모든 항목이 포함됩니다.

---

## 미수금 관리

[계약서](contracts/) 서명 후 결제가 확인되지 않은 건을 추적할 수 있습니다.\
[자동 메시지](announcements/auto-messages.md)의 **미결제 알림**을 활성화하면 자동으로 안내됩니다.
