# 도장 정보

도장의 기본 정보를 설정합니다.\
이 정보는 계약서, 회원 앱, 도장 검색 결과에 표시됩니다.

---

{% hint style="warning" %}
도장 정보를 수정하려면 [지도자 인증](getting-started/leader-verification.md)이 완료되어야 합니다.
{% endhint %}

## 설정 항목

| 항목 | 설명 | 필수 |
|---|---|---|
| 도장 이름 | 공식 도장명 | ✅ |
| 주소 | 도장 위치 (지도 표시용) | ✅ |
| 전화번호 | 대표 연락처 | ✅ |
| 소개글 | 도장 소개 텍스트 | |
| 운영 시간 | 요일별 운영 시간 | |
| 시설 정보 | 보유 시설 (샤워실, 주차장 등) | |
| 도장 사진 | 최대 5장 (3장까지 대표 노출) | |

{% hint style="info" %}
각 항목은 **개별 저장**됩니다. 한 필드를 수정하면 즉시 반영되므로, 한 번에 모든 정보를 입력하지 않아도 괜찮습니다.
{% endhint %}

## 완성도

화면 상단에 **완성도 %**가 표시됩니다. 어떤 정보가 빠져 있는지 한눈에 확인하고, 완성도를 높여가며 설정을 마무리하세요.

## 도장 사진 관리

- 사진은 최대 **5장**까지 업로드할 수 있습니다.
- 드래그로 순서를 변경할 수 있습니다.
- 대표 노출 사진을 최대 **3장**까지 선택할 수 있습니다.
